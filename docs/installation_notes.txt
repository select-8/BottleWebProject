Create Virtual Env:
    C:\Python38\Scripts\virtualenv.exe C:\VirtualEnvs\bottle-project

Install requirments:
    Some libraries are not available via pip and need to be installed from wheels here: https://compassinformatics-my.sharepoint.com/:u:/g/personal/loconnor_compass_ie/EV2-FXNplK1KnRgAVKwI0qgBET0Ub9lv5DltiwF5YczneQ?e=IwJuD2
    Extract the .whl files.
    Update the commented out entries with the path to the extrated wheels
    Activate the venv: C:\VirtualEnvs\bottle-project\Scripts\activate
    Install from requirments.txt: pip install -r requirments.txt

To run the app:
    python app.py
